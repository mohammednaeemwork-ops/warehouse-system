# نظام المخازن - شركة العمار للتوسع العمراني
## تحويل السيستم لتطبيق Windows (.exe)

### المتطلبات
1. نزّل وثبّت **Node.js** من https://nodejs.org (اختار النسخة LTS)
2. تأكد إن التثبيت نجح: افتح **cmd** أو **PowerShell** واكتب:
   ```
   node -v
   npm -v
   ```
   لازم يطلعلك أرقام (مش error)

### الخطوات

**1.** فك ضغط الملف ده في أي فولدر (مثلاً سطح المكتب)

**2.** افتح **cmd** أو **PowerShell** جوه الفولدر ده (Shift + كليك يمين → "Open PowerShell window here")

**3.** ثبّت المكتبات المطلوبة:
   ```
   npm install
   ```
   (هياخد دقيقة أو اتنين أول مرة)

**4.** اعمل الـ build:
   ```
   npm run build:win
   ```

**5.** هتلاقي ملف التثبيت جاهز في فولدر `dist` باسم:
   ```
   نظام المخازن Setup 1.0.0.exe
   ```

**6.** شغّل الملف ده على أي جهاز ويندوز — هيتثبت زي أي برنامج عادي، بأيقونة على سطح المكتب وفي قائمة Start.

---

### ملاحظات
- التطبيق بيتصل بـ Supabase أونلاين، فلازم الجهاز يكون متصل بالإنترنت لتسجيل الدخول واستخدام السيستم.
- لو عملت أي تعديل على `index.html` بعد كده، كرر خطوة `npm run build:win` بس عشان تطلع نسخة جديدة.
- لو حصل أي خطأ أثناء `npm install`، جرب تمسح فولدر `node_modules` (لو اتعمل) وملف `package-lock.json` وكرر الأمر تاني.
